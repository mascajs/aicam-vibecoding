"로컬 PC에서 다운로드 후 더블클릭만으로 바로 실행되는 단일 HTML 파일을 만들어주세요. 코드를 작성할 때 다음 5가지 원칙을 반드시 지켜주세요:
완벽한 단일 파일 (Single HTML): HTML, CSS, JavaScript 코드를 분리하지 말고 하나의 .html 파일 안에 모두 작성해주세요.
프레임워크는 Vue 3 Global 빌드 사용 (React/Babel 금지): 로컬 파일(file://) 환경에서는 React와 Babel을 실시간으로 컴파일하거나 type="module"을 사용하면 브라우저 보안 정책(CORS)에 막혀 빈 화면이 뜹니다. 반드시 변환 없이 바로 실행되는 **Vue 3 (CDN 방식)**를 사용해주세요.
스타일링: Tailwind CSS CDN을 사용하여 디자인해주세요.
보안 정책(API) 예외 처리: 로컬(file://) 환경에서는 크롬 보안상 카메라, 마이크, 웹 시리얼(Web Serial) API 등이 차단됩니다. 해당 기능 사용 전 반드시 try-catch 및 조건문으로 확인하고, 차단된 경우 사용자에게 안내하는 alert() 메시지를 친절하게 띄워주세요.
화면 에러 안전장치 (window.onerror): 예상치 못한 에러로 인해 하얀 빈 화면이 나오는 것을 방지하기 위해, <head> 태그 최상단에 window.onerror를 사용하여 에러 발생 시 화면에 에러 내용이 글자로 출력되도록 안전장치를 추가해주세요."

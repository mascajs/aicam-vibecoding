과일 헥사 대동고 (Fruit Hexa Dash) - GDD

본 문서는 로컬 파일 단일 HTML 환경(Vue 3 CDN)으로 컨버팅된 게임 문서입니다.

  • 장르: 떨어지는 블록 매칭 퍼즐
  • 플랫폼: 데스크톱 브라우저 (단일 HTML 실행 지원)
  • 규칙: 가로, 세로, 대각선 방향으로 같은 과일 3개 이상 매칭 시 제거되며 콤보 연쇄 반응이 일어납니다.
  • 조작: 1인용 (방향키), 2인용 (P1: WASD / P2: 화살표)
  • 대전 모드: 2콤보 이상 발생 시 상대의 맨 아래에 지워지지 않는 돌(🪨) 블록이 밀려 올라갑니다.
